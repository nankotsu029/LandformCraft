# V2-11-06 Measured dimension catalog promotion — Task capsule

（補助資料。正本は `docs/design-v2/task-index.md` / `docs/roadmap.md`）

## 1. Task Index 全文（V2-11-06）

- 状態／Phase／優先度: 未着手／V2-11／P1。
- 目的／前提: `V2-11-04`／`V2-11-05`の実測evidenceに限定してcatalog placement dimension limitを更新する。前提は`V2-11-04`と`V2-11-05`（いずれも2026-07-20完了）。
- Scope／成果物: catalog data／sealed exampleの更新、evidence link（各実測audit）、false-promotion corpus更新（未実測寸法拒否の維持）、README／user／limitations docs同期。
- 非Scope: 実測なし寸法の昇格、LARGE、Release format変更、新feature追加。
- 主変更: built-in catalog data／sealed example、capability matrix docs、README／user／limitations／roadmap。
- 必須test／D/M/S: catalog consistency verifier、sealed example round-trip、昇格entryのevidence link照合、未実測寸法のfalse-promotion拒否回帰、v1回帰。
- Gate: 昇格が実測evidenceの範囲に限定され、catalog検証と対象testが通れば完了。evidenceのない昇格が必要になれば停止する。

## 2. Evidence（昇格の根拠と限界）

| 寸法 | runtime | evidence |
|---|---|---|
| 64×64 | WorldEdit 7.3.19 **および** FAWE 2.15.2 | `V2-6-14`／`V2-6-15` smoke、V2-6 Phase gate |
| 500×500 | FAWE 2.15.2 のみ | `V2-11-04` audit／evidence |
| 1000×1000 | FAWE 2.15.2 のみ | `V2-11-05` audit／evidence |

→ 昇格上限は **1000×1000（FAWE 2.15.2 evidence）**。WorldEdit単独runtimeは 64×64 のまま。1000超（LARGE等）は未実測で拒否を維持。
