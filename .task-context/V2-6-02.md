# V2-6-02 Mutation and effect envelope

## Task Index

- **目的／前提:** final resolverとphysics-sensitive contentからmutation/effect envelopeを保守的に算出する。前提は`V2-6-01`。
- **Scope:** per-tile mutation AABB、union effect envelope、fluid/gravity/neighbor support radius、overflow-safe world bounds、envelope checksum/provenance
- **非Scope:** reservation、snapshot、apply、settle
- **主変更:** `core.v2.placement.envelope`、envelope Schema、fixtures／placement docs
- **必須test:** solid/air/fluid/gravity/boundary fixtures、under-approx corruption、tile/order/thread不変、envelope count/volume/disk estimate admission、world border/Y overflow/unknown physics class拒否
- **Gate:** independent oracleでmutationを包含しeffect上限を安全に証明できれば完了 → `V2-6-03`
