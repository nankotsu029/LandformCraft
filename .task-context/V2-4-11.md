# V2-4-11 Ecology placement

## Task Index

- **目的／前提:** habitat/assemblageとsparse deterministic placement基盤。前提 V2-4-04〜V2-4-10。
- **Scope:** habitat field、assemblage catalog、stable cell/feature seed、density/spacing selector、mangrove/root・coral・alpine vegetation bounded descriptors。
- **非Scope:** cave-local ecology、entity、block entity、Paper placement。
- **主変更:** `model.v2.ecology`、`generator.v2.ecology`、EcologyPlan／catalog Schema、fixtures／docs。
- **必須test:** habitat eligibility／density／spacing／support、dry mangrove／deep-cold coral／unsupported root、whole／tile／seam／thread／candidate順不変、descriptor/spatial-index budget、external script/preset拒否。
- **停止:** dense object gridが必要なら停止。

## Design要点

- EcologyPlan: habitat U16 + closed assemblage catalog + density/spacing kernel
- Sparse: cellHash + spacing lattice winner — no dense placement grid
- Upstream checksum bindings: climate temperature, water wetness/salinity/hydroperiod, snow cover
- ecologyPreset → closed EcologyPreset enum
