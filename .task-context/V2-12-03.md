# Task capsule V2-12-03 — CLI／Paper v2 command routing and E2E

補助資料。正本は `docs/design-v2/task-index.md` §12 と [ADR 0035](../docs/adr/0035-v1-retirement-governance.md) D5／D10。

## 1. Task Index 全文（V2-12-03）

- 目的／前提: CLIとPaperへ正式なv2 command経路（request／design／generate／preview／export／place／undo／status）を追加し、評価用`/lfc r2`を正式経路へ統合する。前提は`V2-12-02`と`V2-6-21`。
- Scope／成果物: CLI subcommand、Paper command＋help／tab completion／permission整合、routing tests、fake gateway E2E（request→export→64×64 placement→Undo）、実機smoke evidence（64×64）。正式名はADR 0035 D5に従い`/lfc v2 <verb>`（`/lfc r2`はdeprecated alias）、CLIは`lfc v2 <verb>`。D10維持verb（`asset`／`doctor`／`job`／`recovery`／`version`／`help`）は本Taskでは変更しない。
- 非Scope: v1 commandの削除・意味変更、D10維持verbのbackend切替（`V2-12-05`）、500／1000、Web UI。
- 主変更: CLI／Paper command adapter、permissions／config、tests、commands／user／admin docs。
- 必須test／D/M/S: command routing／permission／completion回帰、operator／actor／world policyの継承（`V2-6-21` security gate回帰）、v1 command不変回帰、main-thread assertion、correlation ID／stable error code。
- Gate: v2正規経路のE2Eが自動test＋実機smokeで固定されれば完了。

## 2. ADR 0035 の該当決定

- D5: `V2-12-03`は`/lfc v2 <verb>`を正式名として追加し、`/lfc r2 <...>`を同一semanticのdeprecated aliasとして維持、実行時にdeprecation警告を出す。CLIは`lfc v2 <verb>`をsubcommand形式で追加。v1経路は不変（凍結継続）。
- D4: 明示承認された識別子変更は permission node `landformcraft.r2.*`→`landformcraft.v2.*` とcommand名の2件のみ。
- D10: `asset`／`doctor`／`job`／`recovery`／`version`／`help` は削除・変更対象外。

## 3. Verb 表（本Taskの決定）

| canonical | legacy alias | permission | surface |
|---|---|---|---|
| `v2 request validate\|info <request-v2.json>` | — | `landformcraft.v2.request` | CLI＋Paper |
| `v2 design <import\|fixture\|openai\|anthropic> ...` | — | `landformcraft.v2.design` | CLI＋Paper |
| `v2 generate <request> <intent> <exports-root> <release-id> <land\|water> <landY> <waterY>` | — | `landformcraft.v2.export` | CLI＋Paper |
| `v2 export <同上>` | — | `landformcraft.v2.export` | CLI＋Paper |
| `v2 preview <release-path>` | — | `landformcraft.v2.preview` | CLI＋Paper |
| `v2 place plan\|confirm\|execute ...` | `r2 plan\|confirm\|execute` | `landformcraft.v2.{plan,confirm,execute}` | Paperのみ |
| `v2 status <placement-id>` | `r2 status` | `landformcraft.v2.status` | Paperのみ |
| `v2 undo plan\|execute ...` | `r2 undo-plan\|undo-execute` | `landformcraft.v2.undo` | Paperのみ |
| `v2 recover diagnose\|plan\|execute ...` | `r2 recover-*` | `landformcraft.v2.recovery` | Paperのみ |

`generate`は`export`のZIP無し版（directoryのみ）である。CLIからのPaper専用verbは stable error code `V2_PAPER_ONLY` で拒否する。

permission互換: v2 nodeとlegacy r2 nodeのどちらでも通す（`V2-12-06`でr2 nodeを削除）。
