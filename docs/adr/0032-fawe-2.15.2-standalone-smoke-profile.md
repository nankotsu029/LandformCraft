# 0032: FAWE 2.15.2 standalone smoke profile for Release 2

- Status: Accepted
- Date: 2026-07-19
- Task: V2-6-15

## Context

`V2-6-14` は WorldEdit 7.3.19 単独 profile で Release 2 placement smoke を閉じる前提Taskであるが、2026-07-19 re-audit時点では `BLOCKED_EXTERNAL` である。`V2-6-15` は`V2-6-14`のmachine-verifiable evidenceが揃った後、同じ end-to-end／Undo／Recovery smoke を **FAWE 単独** profile で再現し、WorldEdit plugin 併用を禁止する。過去の Phase 6／Beta の FAWE init 記録や v1 smoke を流用してはならない。

## Decision

1. Release 2 FAWE smoke の固定 runtime は次とする。
   - Paper／Minecraft: **1.21.11**
   - FAWE: **FastAsyncWorldEdit-Paper 2.15.2**（build 例: `2.15.2+e9ed0d1`）
   - LandformCraft: 今回 `./gradlew shadowJar` の build
   - Java: **21**
2. `run-fawe/`（または同等の isolated server directory）へ入れる plugin は **FAWE と LandformCraft のみ**。WorldEdit `.jar` の同時導入を拒否する。
3. LandformCraft は FAWE 固有 private API を採用しない。WorldEdit 公開 API 互換経路（既存 `worldedit`／`PaperWorldEditPlacementGateway`）だけを使う。semantic 差や private API 必須が判明したら本 ADR を止め、adapter Task を追加する。
4. smoke 完了証拠は exact FAWE version／LandformCraft JAR SHA-256／checksum／Undo full verify を記録する。server world／secret は Git に入れない。

## Consequences

- dependency／docs／checklist は FAWE **2.15.2** を V2-6-15 の正本 version として参照する。
- `V2-6-15` が `BLOCKED_EXTERNAL` の間も version lock は維持し、別 FAWE version へ暗黙切替しない。
- Beta hardening の未完 FAWE checklist 項目を本 ADR だけで閉じない。

## Alternatives

- 任意の最新 FAWE を都度採用する → 拒否（再現不能）。
- WorldEdit＋FAWE 同時 profile → 拒否（製品規則違反）。
- 過去 FAWE smoke ログの再利用 → 拒否（Task Index 非Scope）。
